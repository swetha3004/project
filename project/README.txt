ApoDoc Symptom Checker is a web-based application that allows users to enter their symptoms and 
receive potential medical conditions based on their input. This project is intended for educational 
purposes and does not include a real medical database or APIs. It serves as a simplified demonstration of a 
symptom checking concept.


Features
Users can enter a list of symptoms.
The application provides a list of potential medical conditions based on the entered symptoms.
A user-friendly interface with minimalistic design.


Technology Stack
HTML
CSS
JavaScript


Disclaimer
This project is not intended for real-world medical use. It does not contain a 
database of actual medical conditions or utilize any APIs for symptom checking. It is purely for
 educational and demonstrative purposes.
